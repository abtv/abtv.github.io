Issue 7: Configuring mongodb

1. Start mongodb
sudo /etc/init.d/mongod start
2. Verifiing that mongodb started successfully
vim /var/log/mongodb/mongod.log
3. Stop mongodb
sudo /etc/init.d/mongod stop

4. to connect to mongodb type
mongo
in command line
5. show dbs - show all databases (in mongo console)
6. db - shows current database
7. use adb - creates database with name adb and switches to adb database


Issue 6: Setting up mongodb

1. Import the public key used by the package management system
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 7F0CEB10
2. Create a list file for MongoDB
echo 'deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen' | sudo tee /etc/apt/sources.list.d/mongodb.list
3. Reload local package database
sudo apt-get update
4. Install the MongoDb packages
sudo apt-get install mongodb-org

Issue 4: Using vim to copy and paste text

Simple commands:
1. Type yy to copy single line
2. Type p to paste text from clipboard
3. Type dd to delete single line (and copy it to clipboard)
4. Type u to undo last change
5. Press Ctrl+R to undo "undo change"

Selection commands:
1. First press Shift+V to select lines and then press y to copy it all
2. Then press p to insert selected lines
