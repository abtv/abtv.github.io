Workflowy

I use Workflowy to organize most of my activity. The main advantage is that I can use Workflowy on my laptop, tablet, and mobile phone and everything is synchronized automatically. The main disadvantage is that you need to use a new tool instead of your favorite IDE, Emacs, Vim, etc.

There are several important things:
  1. I use search by text or tag for navigation.
  2. I use shortcuts for all navigation.
  3. I use a special node called Notes for quick notes. There are no sub-nodes. I put meeting notes during a meeting there or short notes when I'm working on requirements. Then I move it to a more appropriate place: either team's Notion or to Knowledge node.
  4. I use Tasks node for daily and monthly tasks. There are day-nodes and month-nodes. All sub-nodes have a plain list of task-nodes.
  5. I use Projects node for all my projects in-work. There are project-nodes. All sub-nodes have plain task-nodes or idea-nodes. When I am ready to start working on a task I move it to a day-node or a month-node.
  6. I use Ideas node for a long list of ideas. From time to time I get one idea and implement it.
  7. I use Knowledge node for a big collection of short facts and thoughts. I use tags and links to other nodes there.

Keyboard shortcuts:

  1. Cmd+F to search everywhere (sends Cmd+K with Karabiner - I remapped the shortcut).
  2. Escape to search in the scope of a current node.
  3. Cmd+. to zoom in into a current node (makes it full-screen).
  4. Cmd+, to zoom out (goes to a parent node).
  5. Cmd+Shift+J expands current node.
  6. Cmd+Shift+K collapse current node.
  7. Cmd+Enter mark current node as completed.
  8. Cmd+Shift+Backspace delete current node.
