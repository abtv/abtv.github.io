Windows


How to change PuTTY settings

1. Download PuTTY from Downloads page and copy it to C:\Program Files x86\Putty\
1. Window > Appearance, choose ClearType and Lucida Console font, 12-point.
2. Window > Colours and select ANSI Blue in the Select a colour to adjust to Red:74 Green:74 Blue:255 and do the same for ANSI Blue Bold to Red:140 Green:140 Blue:255.

How to install and configure MongoDb

1. Open MongoDb downloads page and download Windows x64 version
2. Then install MongoDb to C:\MongoDb\ by custom installation (all components)
3. Run cmd and type "md \data\db" (this creates default directory \data\db; alternatively you can use C:\mongodb\bin\mongod.exe --dbpath "d:\test\mongo db data")
4. run C:\mongodb\bin\mongod.exe (this starts mongo's main program)
5. run C:\mongodb\bin\mongo.exe (if you like console)
6. install Robomongo from here (if you like applications)
