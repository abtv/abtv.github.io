How to change PuTTY settings

1. Download PuTTY from Downloads page and copy it to C:\Program Files x86\Putty\
1. Window > Appearance, choose ClearType and Lucida Console font, 12-point.
2. Window > Colours and select ANSI Blue in the Select a colour to adjust to Red:74 Green:74 Blue:255 and do the same for ANSI Blue Bold to Red:140 Green:140 Blue:255.

How to install and configure MongoDb

1. Open MongoDb downloads page and download Windows x64 version
2. Then install MongoDb to C:\MongoDb\ by custom installation (all components)
3. Run cmd and type "md \data\db" (this creates default directory \data\db; alternatively you can use C:\mongodb\bin\mongod.exe --dbpath "d:\test\mongo db data")
4. run C:\mongodb\bin\mongod.exe (this starts mongo's main program)
5. run C:\mongodb\bin\mongo.exe (if you like console)
6. install Robomongo from here(if you like applications)

How to install and configure maven

1. Open Maven downloads page and download binary zip.
2. Extract apache-maven***.zip to c:\maven directory.
3. Open "Environmaent variables" and set JAVA_HOME to "C:\Program Files\Java\jdk1.7.0_67" (or your java version).
4. Open "Environmaent variables" and set MAVEN_HOME to "C:\maven".
5. To run maven from command prompt, it is necessary to update the PATH variable with Maven’s installation bin directory. Just Open "Environmaent variables" and append to PATH variable value "C:\maven\bin" (don't forget semicolon before value).
6. Check that all installed correctly "mvn -version"
7. To create jar type "mvn package" in directory where your pom.xml are located.

How to install and configure Light Table

1. Open Leiningen's page, download an installer and install it to c:\.lein. When you have installed Leiningen you can run repl console with cmd.exe. Just type 'lein repl'.
2. Open Light table's page and download binary zip.
3. Extract the zip archive to c:\lighttable directory.
4. Open "Environmaent variables" and append to PATH to "C:\Program Files\Java\jdk1.7.0_67\bin" (or where your javac file is located).
