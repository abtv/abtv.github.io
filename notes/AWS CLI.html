AWS CLI

Configure AWS CLI aliases (add them to your .zshrc file)

alias aws-config="aws configure sso"
alias aws-login="aws sso login --profile ProfileName"
alias aws-creds="vi ~/.aws/credentials"

Configure SSO

Run aws-config and follow on-screen instructions.

Configuration is stored in ~/.aws/config file.

Login with SSO

Run aws-login to renew expired AWS token.

Access to RDS

Log in to AWS, copy, run aws-creds and paste the following fields.

aws_access_key_id
aws_secret_access_key
aws_session_token
