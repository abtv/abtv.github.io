AWS CLI

Configure SSO

Run the following command and follow on-screen instructions.

aws configure sso

Configuration is stored in ~/.aws/config file.

Login with SSO

Use the following bash alias to renew expired AWS token.

aws-login='aws sso login --profile ProfileName'

Access to RDS

Log in to AWS, copy, and paste the following fields to ~/.aws/credentials file.

aws_access_key_id
aws_secret_access_key
aws_session_token
