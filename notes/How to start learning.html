How to start learning

Important questions

Before starting learning I try to answer the following questions in this order:

  1. Why should I learn?
  2. What should I learn?
  3. How should I learn?

Without answering these questions (in this order!) it's very likely I will spend my time in vain.

Important notes

  • Watching videos on YouTube usually is not learning. Learning changes our behavior, watching videos doesn't.
  • Reading books usually is not learning if I don't practice what I learned.
  • If I don't practice what I learned I will forget it.
