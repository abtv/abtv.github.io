1. Types

Every object has a type. Type defines what we can do with the object.

  • Physical object: has 4d representation in space and time, object is finite.
  • Class: a set of objects which are grouped using some property value.
  • Tuple: it's 3 items: object, relation, another object. Order is important.

3. Explanation

Desired properties of an explanation:

  1. It is close to reality: we can explain something with narrow classes or physical objects.
  2. It has details.
  3. Refutability: we know in which circumstances our explanation will be false. The more such ways we find the better. To find the ways we need to answer the following question: what should happen to make our explanation false?

Mnemonic technique: it's easier to remember facts if we put the facts into a narrative (story): somebody doing something. Something is your facts.
