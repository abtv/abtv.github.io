WebStorm

There are several principles I follow:

  1. Use Vim mode without relative numbers.
  2. Use one-keystroke keybindings.
  3. Use distraction-free mode for documenting and blogging.
  4. Use soft wraps.
  5. Open WebStorm with "ws" alias.
  6. I use Git CLI interface and Tig.

One-keystroke keybindings

  1. Cmd+o Open recent project
  2. Cmd+p Toggle project
  3. Cmd+; Search everywhere
  4. Cmd+[ Go to the left tab
  5. Cmd+] Go to the right tab
  6. Cmd+W Close current tab
  7. Cmd+E Select and expand
  8. Cmd+B Git blame
  9. Cmd+T Run tests

Open WebStorm with "ws" alias.

Add the following to your .zshrc/.bashrc file.

alias ws="webstorm ."
