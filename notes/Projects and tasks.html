Projects and tasks

My approach to projects and tasks is inspired by David Allen's GTD system.

Key principles

There are 3 key principles I follow:

  1. I have a written list of projects and areas of interest. Every day I read the list and think which next action I can do to be closer to finishing a project or maintain an area of interest. I mark such next actions with #today, #tomorrow, or #later tag.
  2. I have a written list called Inbox to save all purchases, interesting articles, videos, etc. Every day I check the Inbox list and either move an item to some project or delete it.
  3. If I'm not sure that I want to do something I add #idea tag to the item.

Tool

I use Workflowy → application to work with the lists I described above. There are the following useful features:

  1. Workflowy is installed on my laptop, mobile and tablet. It is synchronized automatically. I can use it everywhere.
  2. Workflowy allows to search by a tag. It means I can create a quick view with daily tasks or a similar view.
  3. Workflowy allows to focus on a list and show only the list's items.
  4. I can use hotkeys to work with Workflowy. Usually, I don't use a mouse/touchpad.

Keyboard shortcuts

  1. Cmd+F to search everywhere (sends <Cmd+K> with Karabiner - I remapped the shortcut).
  2. Escape to search in the scope of a current node.
  3. Cmd+. to zoom in into a current node (makes it full-screen).
  4. Cmd+, to zoom out (goes to a parent node).
  5. Cmd+Shift+J expands current node.
  6. Cmd+Shift+K collapse current node.
  7. Cmd+Enter mark current node as completed.
  8. Cmd+Shift+Backspace delete current node.
