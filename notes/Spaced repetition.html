Spaced repetition

The problem

You learn new things and want to remember more.

Solution

The best way to remember something is to use it immediately. If it's not possible you can use spaced repetitions: recall or explain aloud what you learned. You need to do it in 1, 2, 3, 5, 8, 13 days.
