Operating system

What operating system is

As an abstraction consists of the following 3 items:

  1. Planner of processes (may be distributed across machines).
  2. Memory management (may be distributed across machines).
  3. Means of interprocess communications.

Driver provides an abstraction to not think how exactly device works.

Process vs thread

Threads share the same address space, processes don't.

Interruption

Interruption is a way how user code may return control to operating system and vice versa. Exception also generates an interruption.
